# Research automatic cloud recognition

## Summary

Investigate a later camera and uploaded-photo recognition feature without
weakening privacy, scientific uncertainty, or the teaching-first purpose.

## Acceptance Criteria

- [ ] Compare local and hosted inference approaches.
- [ ] Define privacy and consent behavior.

- [ ] Specify how recognition supports learning instead of replacing it.

# Scientific and Image Sources

## Source Policy

Every learner-facing scientific claim belongs to a source record with:

- source organization and document title;
- stable URL and retrieval date;
- claim scope;
- whether the source is normative, operational guidance, or explanatory;
- a short plain-language reason the source is trustworthy.

Photographs additionally record:

- creator;
- license and required attribution;
- original file page;
- whether the photograph is used as identification evidence or decoration;
- any crop or color adjustment.

No WMO photograph is copied into the application without explicit reuse
permission. WMO terminology and classification are paraphrased with links to
the International Cloud Atlas.

## Primary References

1. World Meteorological Organization, *International Cloud Atlas*:
   <https://cloudatlas.wmo.int/>
2. EASA, *Easy Access Rules for Aircrew*, meteorology syllabus:
   <https://www.easa.europa.eu/en/document-library/easy-access-rules/easy-access-rules-aircrew-regulation-eu-no-11782011>
3. FAA, *Aviation Weather Handbook FAA-H-8083-28A*:
   <https://www.faa.gov/sites/faa.gov/files/FAA-H-8083-28A_FAA_Web.pdf>
4. ICAO/WMO operational codes are introduced through official or national
   aviation-meteorology guidance and linked at the lesson level.
5. Windy help/community documentation is used only to explain the interface
   concepts users encounter. The application does not embed or reproduce Windy.

## Preferred Photograph Pools

- NOAA and US National Weather Service public-domain material where the
  individual asset page confirms status.
- NASA material under NASA media usage guidance.
- Wikimedia Commons assets whose individual file pages state compatible
  Creative Commons or public-domain terms.

Each selected image is audited individually; the reputation of a host site
does not substitute for a license check.
